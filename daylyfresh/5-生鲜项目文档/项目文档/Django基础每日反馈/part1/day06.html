day06反馈

  • 1.了解Django的admin后台管理页面的基本使用。

  • 2.掌握Django中文件的上传。

    首先创建上传文件目录并设置MEDIA_ROOT配置项。

后台管理页面上传:
1) 创建模型类，在模型类中图片对应的字段使用ImageField字段类型。
class Goods(models.Model):
  # ...
  image = models.ImageField(upload_to='相对保存目录')
  # ...
2) 迁移生成表并注册模型类。
3) 通过admin后台管理页面选择图片点击保存即可实现上传。

自定义页面上传:
1) 页面上使用表单并把method设置'post'，同时enctype设置为'multipart/form-data'。
<form method='post' enctype='multipart/form-data' action='/upload_pic'>
  ...
  <input type='file' name='pic'/>
  <input type='submit' value='上传'/>
</form>
2) 在后台对应的处理视图中接收文件并保存。
request.FILES：保存用户上传的文件信息。

  • 3.掌握Django中的分页操作。

    分页类: Paginator
模块: from django.core.paginator import Paginator
使用: paginator = Paginator('数据集', '每页数目')
方法:
  1) page('数字'): 返回值为Page类的对象，包含某一页上面的数据信息。
属性:
  1) num_pages: 分页之后的总页数。
  2) page_range: 分页之后页码的列表。

Page类:
方法:
  1) has_previous: 判断当前页是否有上一页。
  2) has_next: 判断当前页是否有下一页。
  3) previous_page_number: 获取上一页的页码。
  4) next_page_number: 获取下一页的页码。
属性:
  1) number: 获取当前页的页码。
  2）paginator: 和Page对象相关的Paginator对象。
